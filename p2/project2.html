My website introduces abstract English letterforms with CSS code and shows the interaction between Korean letters, Hangul based on a system called "Romanization of Korean" Romaja literally means Roman letters in Korean, and refers to the Latin script. â€” it refers to writing Korean words using the Latin script: either romanizing individual words in a Korean text or writing an entire Korean text in the Latin script.

To make the connection, when you hover the English letter it transforms into White Korean letterform, which refers that those two letters have a similar sound to each other. If a letter returns back as black, not white color, it means it doesn't have any sound match with one single Korean character. For instance, when writing letter 'C' sound or literally just letter 'C', it needs to combine Korean Consonant and vowel to make sound 'C'. But there are some Korean letters that sound similar; A letter 'A', you only need one single Korean letter, either vowel or consonant that makes English, 'A' sound.

Nemo

Letter forms on the left side represent numbers, 0 to 9 in order. The rest two are Exclamation mark and quatation mark.

My font name is Nemo, which means square in Korean. The letterforms are all fit into the same size square, which proposition is 8:9. So all the width and the height are the same. I was inspired by Bauhaus font so they look pretty abstract and feels heavy. Also, elements of letterforms are composed of basic shapes: square, circle, and triangle. You can also see those elements on my website.
